Detailed Description

La classe FileSplitter ouvre un fichier .SCVLOG et le découpe en morceaux plus petits de 20.000 messages SOAP chacun (avec un maximum de 10 fichiers). Ceci est utile quand le fichier SVCLOG est tellement gros qu'il ne peut pas être ouvert par les outils d'analyse. Les nouveaux fichierx XML sont nommés: xxx.n.svclog.

La classe transforme un fichier SVCLOG en un fichier XML. Elle lit le fichier SVCLOG caractère-par-caractère et n'utilise PAS de parser XML.
