Detailed Description

Cette classe gére la liste des messages SOAP échangés. Tous les membres de la classe sont statiques: Pas besoin d'instancier la classe.

void Messages::setAction ( std::string *  action)
static

Cette méthode analyse la string fourni et extrait les deux derniers mots significatifs, pour les enregister dans les champs Message.module et Message.action. Exemple "Session" -> "OpenOrReuse".

Attention
le paramètre action est modifié pour ne laisser que le dernier item de la string.
Parameters
[in,out]actionUne url du type "http://www.sgt.eu/VEDA.SOA/Session.v1/Session/OpenOrReuse".
