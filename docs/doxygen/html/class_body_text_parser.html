Detailed Description

La classe statique BodyTextParser analyse le Body XML d'un message, pour l'afficher sous forme de texte simple dans l'onglet "Text View".

Elle sert à convertir un blob XML (typiquement le body d'un message SOAP) en une string formatée, contenant des balises de coloration syntaxique, pour un affichage dans QT. Le parser SAX est basé sur la librarie EXPAT.

David de Lorenzo
