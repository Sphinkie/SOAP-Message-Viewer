Detailed Description

La classe FileRepair ouvre un fichier .SCVLOG dont des messages sont tronqués au début ou à la fin, et essaye de le réparer. Un nouveau fichier XML est généré (avec l'extension ".repaired.xml" ), et contenant le flux réparé.

La classe transforme un fichier SVCLOG en un fichier XML. Elle lit le fichier SVCLOG caractère-par-caractère et n'utilise PAS de parser XML.
