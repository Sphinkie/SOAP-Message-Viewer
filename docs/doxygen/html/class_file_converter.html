Detailed Description

FileConverter transforme un fichier .SCVLOG en un fichier .FASTER.XML en supprimant des balises inutiles, ce qui accélère les futurs parsings du fichier.

La classe FileConverter utilise le parser XML Expat qui est open-source, très rapide, et SAX (parsing au fil de l'eau). Elle gère les gros fichiers grace à un système de buffer.
